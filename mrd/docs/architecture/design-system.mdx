# Design System

## Overview

MEDrecord uses **Mantine** as the UI framework - the same system Medplum uses natively. This ensures sync-safe theming and consistent design across all brands.

## Theming Architecture

### Single Source of Truth

Each brand has **one** Mantine theme file:

```
mrd/apps/
├── healthtalk/src/theme.ts    # HealthTalk theme
├── coachi/src/theme.ts        # Coachi theme
├── medsafe/src/theme.ts       # MedSafe theme
└── medrecord/src/theme.ts     # MEDrecord theme
```

No CSS files, no OKLCH, no runtime generation - just Mantine's native `createTheme()`.

## Theme Structure

### Mantine Theme File

```typescript
// mrd/apps/healthtalk/src/theme.ts
import { createTheme, MantineColorsTuple } from '@mantine/core';

const healthtalkBlue: MantineColorsTuple = [
  '#E6F0FF',  // 0 - lightest
  '#CCE0FF',  // 1
  '#99C2FF',  // 2
  '#66A3FF',  // 3
  '#3385FF',  // 4
  '#0066CC',  // 5 - primary (default)
  '#0052A3',  // 6
  '#003D7A',  // 7
  '#002952',  // 8
  '#001429',  // 9 - darkest
];

export const healthtalkTheme = createTheme({
  primaryColor: 'healthtalkBlue',
  colors: {
    healthtalkBlue,
  },
  fontFamily: 'Inter, -apple-system, BlinkMacSystemFont, Segoe UI, sans-serif',
  headings: {
    fontFamily: 'Inter, -apple-system, BlinkMacSystemFont, Segoe UI, sans-serif',
    fontWeight: '600',
  },
  defaultRadius: 'md',
});
```

### Applying Theme

```typescript
// mrd/apps/healthtalk/app/layout.tsx
import { MantineProvider } from '@mantine/core';
import { healthtalkTheme } from '../src/theme';
import '@mantine/core/styles.css';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="nl">
      <body>
        <MantineProvider theme={healthtalkTheme}>
          {children}
        </MantineProvider>
      </body>
    </html>
  );
}
```

## Brand Color Palettes

Each brand uses a 10-shade color palette (Mantine standard):

| Brand | Primary | Hex (shade 5) | Use Case |
|-------|---------|---------------|----------|
| **HealthTalk** | Blue | `#0066CC` | Trust, healthcare, communication |
| **Coachi** | Green | `#2E7D32` | Growth, wellness, coaching |
| **MedSafe** | Teal | `#00796B` | Safety, medication, reliability |
| **MEDrecord** | Indigo | `#3F51B5` | Professional, enterprise, core |

### Color Shade Usage

| Shade | Usage | Example |
|-------|-------|---------|
| 0-2 | Backgrounds, hover states | Light backgrounds |
| 3-4 | Borders, secondary elements | Input borders |
| **5** | **Primary color** | Buttons, links |
| 6-7 | Hover/active states | Button hover |
| 8-9 | Text on light backgrounds | Dark text |

## Shared Components (@mrd/ui)

The `@mrd/ui` package provides shared components that work with any Mantine theme:

```typescript
// Using shared component
import { DataTable } from '@mrd/ui/components/data-table';

// Component automatically inherits brand theme from MantineProvider
<DataTable data={patients} columns={columns} />
```

## Accessibility

### Color Contrast

All brand color palettes are designed to meet WCAG 2.1 AA:

- Primary text (shade 5 on white): minimum 4.5:1
- Interactive elements: minimum 3:1

### Dark Mode

Mantine handles dark mode automatically:

```typescript
<MantineProvider 
  theme={healthtalkTheme}
  defaultColorScheme="auto"  // Respects system preference
>
```
