# Channel D: Email (Postmark)

## Overview

Email delivery sends a branded HTML email with a magic link to the web form. This is the most formal channel and serves as a fallback when other channels are unavailable.

## Flow

```
1. Task created with channel = "email"
2. Gateway generates magic link
3. Gateway sends email via Postmark
4. Patient receives email
5. Patient clicks magic link
6. Web form opens (Channel A)
7. Patient completes questionnaire
8. Task marked complete
```

## Postmark Integration

```typescript
// lib/email/send.ts
import { ServerClient } from 'postmark';

const client = new ServerClient(process.env.POSTMARK_SERVER_TOKEN!);

export async function sendQuestionnaireEmail(
  to: string,
  data: EmailTemplateData
) {
  return client.sendEmailWithTemplate({
    From: process.env.POSTMARK_FROM_EMAIL!,
    To: to,
    TemplateAlias: 'questionnaire-request',
    TemplateModel: {
      patient_first_name: data.patientFirstName,
      practitioner_name: data.practitionerName,
      organization_name: data.organizationName,
      questionnaire_title: data.questionnaireTitle,
      estimated_duration: data.estimatedDuration,
      deadline: formatDate(data.deadline, 'nl'),
      magic_link_url: data.magicLinkUrl,
    },
    MessageStream: 'outbound',
  });
}
```

## Reminder Email

Sent 24 hours before deadline if not completed:

```
Herinnering: Vragenlijst nog niet ingevuld

Beste {{patient_first_name}},

U heeft nog geen antwoord gegeven op de vragenlijst "{{questionnaire_title}}".

De deadline is morgen ({{deadline}}).

[Vragenlijst invullen]
```

## Tracking

| Event | Postmark Webhook | Action |
|-------|------------------|--------|
| Delivered | delivery | Update Task businessStatus |
| Opened | open | Log event (optional) |
| Link clicked | click | Log event (optional) |
| Bounced | bounce | Mark delivery failed, alert |
| Spam complaint | spam | Unsubscribe patient |

## Error Handling

| Scenario | Action |
|----------|--------|
| Invalid email | Skip email, try other channel |
| Bounce | Mark failed, notify clinician |
| Spam complaint | Add to suppression list |
| Rate limited | Queue for retry |
