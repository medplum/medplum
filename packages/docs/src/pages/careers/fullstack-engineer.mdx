# Full Stack Software Engineer

![Full Stack Software Engineer](/img/careers/fullstack-engineer.webp)

## About the Role

Join our rapidly growing healthcare technology startup as we build scalable, elegant solutions that serve millions of patients. As a Full Stack Software Engineer, you'll work across our entire technology stack, from crafting intuitive user interfaces to designing robust backend services. We're looking for a versatile engineer who can move fluidly between frontend and backend development, someone who's excited about owning features end-to-end and making a direct impact on healthcare delivery. This is an opportunity to work with modern technologies while solving meaningful challenges in healthcare.

### What You Will Do

- Build end-to-end features across our React frontend and Node.js backend
- Design and implement RESTful APIs that power our healthcare applications
- Collaborate with product and design teams to create intuitive user experiences
- Optimize application performance and ensure scalability across the stack
- Contribute to our architecture decisions and technical direction
- Work with healthcare data while maintaining HIPAA compliance and security

## About You

- You hold a 4-year degree in computer science, software engineering, or a related field
- You bring 4+ years of professional software development experience across frontend and backend
- You have strong experience with TypeScript, React, and Node.js
- You're comfortable with SQL databases and API design
- You take pride in writing clean, well-tested, maintainable code
- You have a proven track record of shipping production features end-to-end
- You are based in the SF Bay Area and are ready for twice-weekly in-person collaboration

#### Bonus

- Experience with healthcare systems or HIPAA compliance
- Knowledge of AWS or other cloud platforms
- Experience with PostgreSQL or distributed databases
- Previous experience at an early-stage startup
- Strong understanding of security best practices
