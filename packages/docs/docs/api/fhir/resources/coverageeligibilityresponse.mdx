<div> <a name="scope"></a> <p> The <Link to="/docs/api/fhir/resources/coverageeligibilityresponse">CoverageEligibilityResponse</Link> resource provides eligibility and plan details from the processing of an  <Link to="/docs/api/fhir/resources/coverageeligibilityrequest">CoverageEligibilityRequest</Link> resource. It combines key information from a payor as to whether a <Link to="/docs/api/fhir/resources/coverage">Coverage</Link> is  in-force, and optionally the nature of the Policy benefit details as well as the ability for the insurer to indicate whether  the insurance provides benefits for requested types of services or requires preauthorization and if so what supporting information may be required. </p> <p>The <Link to="/docs/api/fhir/resources/coverageeligibilityresponse">CoverageEligibilityResponse</Link> resource is a "event" resource from a FHIR workflow perspective -
