    <div> <a name="scope"></a>  <p> The <a href="/docs/api/fhir/resources/devicemetric">DeviceMetric</a> resource describes mandatory static properties that characterize a direct or derived, quantitative or qualitative biosignal measurement, setting, or calculation produced by a medical device.  The <a href="/docs/api/fhir/resources/devicemetric">DeviceMetric</a> resource can also be used to describe the non-static but highly relevant properties to the metric such as metric status, metric last calibration time and type, measurement mode, color, reference link to the parent DeviceComponent to where it belongs, and any capabilities that the metric offers (for example: setting the metric label). </p> <p> Note: </p> <p> For the initial scope, this <a href="/docs/api/fhir/resources/devicemetric">DeviceMetric</a> resource is only applicable to describe a single metric node in the containment tree that is produced by the context scanner in any medical device that implements or derives from the ISO/IEEE 11073 standard. </p> </div>
