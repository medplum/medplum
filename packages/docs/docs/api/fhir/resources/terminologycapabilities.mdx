# TerminologyCapabilities

A <Link to="/docs/api/fhir/resources/terminologycapabilities">TerminologyCapabilities</Link> resource documents a set of capabilities (behaviors) of a FHIR Terminology Server that may be used as a statement of actual server functionality or a statement of required or desired server implementation.
