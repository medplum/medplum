# EffectEvidenceSynthesis

The <Link to="/docs/api/fhir/resources/effectevidencesynthesis">EffectEvidenceSynthesis</Link> resource describes the difference in an outcome between exposures states in a population where the effect estimate is derived from a combination of research studies.
