# EffectEvidenceSynthesis

The <a href="/docs/api/fhir/resources/effectevidencesynthesis">EffectEvidenceSynthesis</a> resource describes the difference in an outcome between exposures states in a population where the effect estimate is derived from a combination of research studies.
