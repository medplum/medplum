<div> <a name="bnc"></a> <p><strong>This resource is a <em>definition</em> resource from a FHIR workflow perspective</strong> - see <a href="https://www.hl7.org/fhir/workflow.html">Workflow</a>, specifically <a href="https://www.hl7.org/fhir/workflow.html#definition">Definition</a>.</p> <p>The scope of the <Link to="/docs/api/fhir/resources/evidencevariable">EvidenceVariable</Link> resource is to describe a "PICO" element that knowledge (evidence, assertion, recommendation) is about. PICO stands for: <Link to="/docs/api/fhir/datatypes/population">Population</Link> - the population within which exposures and outcomes are being considered Intervention - the conditional state or exposure state being described for which one is expressing the risk of or effect on an outcome Comparison - the alternative conditional state or alternative exposure state when one is comparing states for the effect on an outcome Outcome - the result or effect of an intervention or exposure in a population PICO (and its variants like PECO or PICOT) is a universal acronym used in evidence-based medicine communities to clearly express research questions and evidence findings. The <Link to="/docs/api/fhir/resources/evidencevariable">EvidenceVariable</Link> resource allows expression of the components of a PICO question in codeable and reusable formats.</p> </div>
