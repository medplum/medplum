# CoverageEligibilityRequest

The <a href="/docs/api/fhir/resources/coverageeligibilityrequest">CoverageEligibilityRequest</a> provides patient and insurance coverage information to an insurer for them to respond, in the form of an <a href="/docs/api/fhir/resources/coverageeligibilityresponse">CoverageEligibilityResponse</a>, with information regarding whether the stated coverage is valid and in-force and optionally to provide the insurance details of the policy.
