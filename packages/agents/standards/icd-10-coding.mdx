# ICD-10 Coding

## Overview

International Classification of Diseases, 10th Revision (ICD-10) is the
standard coding system for diagnoses and health conditions.

**Version:** ICD-10-CM 2024
**Publisher:** WHO / CMS
**System URI:** `http://hl7.org/fhir/sid/icd-10-cm`

## Code Structure

ICD-10 codes follow a hierarchical structure:

```
A00.0
│││ └── Extension (4th+ characters)
││└──── Category (3rd character)
│└───── Block (2nd character)
└────── Chapter (1st character)
```

## FHIR Mapping

ICD-10 codes are used in FHIR `Condition` resources:

```json
{
  "resourceType": "Condition",
  "code": {
    "coding": [
      {
        "system": "http://hl7.org/fhir/sid/icd-10-cm",
        "code": "I21.0",
        "display": "ST elevation (STEMI) myocardial infarction of anterior wall"
      }
    ]
  }
}
```

## Coding Guidelines

1. **Specificity** - Code to the highest level of specificity
2. **Primary diagnosis** - List the main condition first
3. **Manifestations** - Code underlying disease before manifestation
4. **Combination codes** - Use single code when available
5. **Sequencing** - Follow official coding guidelines
